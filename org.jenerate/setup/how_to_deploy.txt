How to deploy!!
- Increment versions of Jenerate plugin, feature and site
- Opening the feature, click the little "export as a deployable feature" button, choose a directory and finish
- Clone the github.io project on eclipse (import as general project)
- Add the previous update site from the root of "update" to a new folder corresponding to the version
- Add the new update site to the root of "update"
- Edit the github.io index.html and the jenerate README.md to update the changelog
- commit and verify that it works like a charm
- Do not forget to update the eclipse market place
